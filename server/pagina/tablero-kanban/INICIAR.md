# Iniciar el proyecto

## Requisito
Node.js v18 o superior

## Pasos

```bash
# 1. Ir a la carpeta del proyecto
cd /home/elvis/Documentos/VScode/pagina

# 2. Iniciar el servidor
npm start
```

## Abrir en el navegador
- **Tablero**: http://localhost:7000
- **Completados**: http://localhost:7000/completados.html

## Detener
Presiona `Ctrl + C` en la terminal.
